== Installation ==

This section describes how to install the plugin and get it working.

1. Install the wordpress plugin first
You can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder

2. Activate the plugin

3. You will see "YT Video Slider" settings from right side of your wordpress menu.

Configuration Options:

Youtube Username : Put your Youtube Username

Youtube Video URL : Put the video URL https://www.youtube.com/watch?v=qUGnu0gXtn4


qUGnu0gXtn4 This part only on the field you have to put only

Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.
